# Travel Blog

Веб-приложение для обмена опытом путешественников. Пользователи могут оставлять комментарии, делиться фотографиями и просматривать комментарии других пользователей. Проект написан на Vue и TypeScript с использованием Vue Router, Pinia, библиотек Axios, препроцессора SCSS и с помощью существующего API.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
